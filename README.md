# smemsc.github.io

GitHub Pages landing site for the Seven Mountains EMS Council (SMEMSC).

## What It Is

This repository hosts the static landing page for the SMEMSC GitHub organization. It provides a modern, responsive overview of the council's mission, services, coverage area, and contact information for visitors arriving via GitHub.

### Features

- **Responsive design** — optimized for desktop, tablet, and mobile
- **Modern CSS** — custom properties, smooth animations, and accessible focus states
- **Comprehensive overview** — services, counties served, contact details, and quick stats
- **Performance-focused** — single-file HTML with embedded CSS, no external dependencies beyond fonts

## Setup & Usage

This is a static site automatically deployed via [GitHub Pages](https://pages.github.com/).

```bash
# Clone for local editing
git clone https://github.com/smemsc/smemsc.github.io.git

# Preview locally
open index.html
# or
python3 -m http.server 8000
```

Push changes to the main branch to update the live site.
